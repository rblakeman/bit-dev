Welcome to the Bit Quick-Start!
This tutorial will help you learn how to get started with Bit, and build your first composable software.
These Docs were built using Bit components. Following this will show you how simple it is to build even a complicated app, when using components.

## Introduction

In this quick start you will learn how to install Bit, open a new workspace, and build a Wiki-app using components we have built for you!
A Wiki-app is a great way to align a team, share information and communicate inside any organization. Building a Wiki-app using components allows many team to collaborate and participate.
