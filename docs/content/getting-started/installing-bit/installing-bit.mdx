# Installing Bit

Run the following to install Bit on your machine:

```bash
npx @teambit/bvm install
```

The output is similar to the following:

```bash
version 0.0.712 was installed on /Users/jane/.bvm/versions/0.0.712
```

For troubleshooting, see [Troubleshooting](https://bit.dev/docs/reference/using-bvm#troubleshooting.).

## Updating Bit

Run the following to update to the latest version of Bit:

```
npx @teambit/bvm upgrade
```
